# Building a Stock Analysis and Alert System on AWS

This workshop guides the deployment of a stock analysis system using AWS Serverless architecture. The system collects market data from Yahoo Finance, stores raw data in Amazon S3, processes events asynchronously through Amazon SQS and AWS Lambda, calls Amazon Bedrock for AI-assisted analysis, and stores reports in Amazon DynamoDB for display on the Dashboard.

The main goal of this workshop is to help learners understand how AWS managed/serverless services can be combined to build a scalable financial data processing pipeline with cost control and a Human-in-the-loop review process before recommendations are sent to customers.
